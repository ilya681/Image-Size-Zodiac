## 🌟 درباره پروژه
این ابزار به شما امکان می‌دهد حجم فایل تصاویر خود را به شدت کاهش دهید بدون اینکه کیفیت قابل مشاهده‌ای از دست برود. تمام عملیات در مرورگر شما انجام می‌شود، بنابراین تصاویر شما هرگز به سرور ما ارسال نمی‌شوند – امن و خصوصی.

**نام مخزن (`Image-Size-Zodiac`) صرفاً یک اسم قدیمی است و ربطی به طالع‌بینی ندارد. این پروژه یک ابزار کاربردی برای کاهش حجم عکس است.**

## ✨ ویژگی‌ها
- 📉 کاهش حجم تا ۸۰٪ (مثلاً از ۵ مگابایت به ۱ مگابایت)
- 🖼️ حفظ کیفیت بصری – بدون پیکسل‌شدگی یا تاری
- ⚡ سرعت بالا – پردازش لحظه‌ای در مرورگر
- 🔒 کاملاً خصوصی – تصاویر هرگز آپلود نمی‌شوند
- 📱 طراحی واکنش‌گرا (responsive) – عالی در موبایل و دسکتاپ
- 🌙 تم دارک اختصاصی و جذاب
- 🧩 پشتیبانی از فرمت‌های JPG، PNG، BMP، WebP

## 🚀 نحوه استفاده
1. روی دکمه «آماده‌ای شروع کنی! پس بزن بریم» کلیک کنید.
2. عکس خود را آپلود کنید.
3. کیفیت مورد نظر را با اسلایدر تنظیم کنید.
4. روی دکمه «کاهش حجم عکس» کلیک کنید.
5. تصویر فشرده شده را دانلود کنید.

> نکته: هرچه کیفیت کمتر باشد، حجم فایل کمتر می‌شود. معمولاً کیفیت ۷۵٪ بهترین تعادل بین حجم و کیفیت است.

## 🛠️ تکنولوژی‌های استفاده شده
- HTML5
- CSS3 (ریسپانسیو، دارک تم، انیمیشن)
- JavaScript (ES6+)
- Canvas API برای فشرده‌سازی تصاویر
- میزبانی روی **GitHub Pages**
