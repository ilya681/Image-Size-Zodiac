## 🎯 این ابزار چه کاری انجام می‌دهد؟
این ابزار حجم فایل تصاویر شما را **به شدت کاهش می‌دهد** بدون اینکه کیفیت قابل مشاهده‌ای از دست برود. فرقی نمی‌کند عکس محصول، عکس پرسنلی یا تصاویر وبلاگتان باشد – با استفاده از الگوریتم‌های بهینه، حجم فایل را تا ۸۰٪ کم می‌کند در حالی که چشم انسان تفاوتی نمی‌بیند.

## ✨ ویژگی‌های کلیدی
- 📉 **کاهش حجم عکس** تا ۸۰٪ (مثلاً از ۵ مگابایت به ۱ مگابایت)
- 🖼️ **حفظ کیفیت بصری** – بدون پیکسل‌شدگی یا تاری
- ⚡ **سرعت بالا** – پردازش در مرورگر شما، بدون نیاز به آپلود روی سرور
- 🔒 **امن و خصوصی** – تصاویر هرگز سرور ما را نمی‌بینند (همه چیز در دستگاه شما انجام می‌شود)
- 📱 **طراحی واکنش‌گرا** – در موبایل، تبلت و دسکتاپ عالی کار می‌کند
- 🌓 **حالت تاریک/روشن** (اختیاری – اگر در سایت خود دارید)

## 🚀 نحوه استفاده
1. روی دکمه **«انتخاب تصویر»** کلیک کنید.
2. یک عکس (با فرمت JPG، PNG یا WebP) را انتخاب کنید.
3. ابزار به‌طور خودکار حجم آن را کاهش می‌دهد.
4. دکمه **«دانلود»** را بزنید تا عکس فشرده‌شده را ذخیره کنید.

> نکته: هرچه عکس بزرگ‌تر باشد، کاهش حجم چشمگیرتر خواهد بود.

## 🛠️ تکنولوژی‌های استفاده شده
- HTML5
- CSS3 (با قابلیت ریسپانسیو و دارک‌مود)
- JavaScript (ES6+)
- کتابخانه‌ی `browser-image-compression` (یا هر کتابخانه‌ای که استفاده می‌کنید – می‌توانید نام واقعی را جایگزین کنید)
- میزبانی روی **GitHub Pages**
