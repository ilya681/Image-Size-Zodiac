# 📐 Image Size Zodiac – ابزار هوشمند تعیین اندازه تصویر با طالع‌بینی

> ابزاری رایگان، سریع و متن‌باز که با ترکیب `image size` و `zodiac`، اندازه ایده‌آل تصاویر را بر اساس ماه تولد شما تعیین می‌کند.

## ✨ درباره پروژه
**Image Size Zodiac** راه‌حلی نوآورانه برای تعیین **اندازه استاندارد تصاویر** است. این ابزار کاملاً رایگان، به شما کمک می‌کند تا بر اساس **ماه تولد** خود، ابعاد بهینه تصاویر را برای طراحی گرافیک، توسعه وب و شبکه‌های اجتماعی پیدا کنید.

## 🎯 کلمات کلیدی پروژه
`ابزار اندازه تصویر`, `image size calculator`, `zodiac image tool`, `سایز استاندارد عکس`, `resize image by birth month`

## ✨ ویژگی‌های کلیدی
- 🔮 محاسبه **سایز تصویر** شخصی‌سازی شده با طالع‌بینی
- 🚀 سرعت بالا، بدون نیاز به ثبت‌نام یا سرور
- 📱 طراحی واکنش‌گرا (Responsive) و سازگار با موبایل
- 🔍 بهینه شده برای موتورهای جستجو (SEO-Friendly)
- 🌐 دسترسی رایگان و همیشگی

## 🚀 نحوه استفاده
1. وارد سایت ابزار شوید.
2. ماه تولد خود را از منوی کشویی انتخاب کنید.
3. روی دکمه **"دریافت سایز"** کلیک کنید.
4. سایز پیشنهادی مخصوص ماه تولدتان را مشاهده خواهید کرد.

## 🛠️ تکنولوژی‌ها
- HTML5
- CSS3 (شامل گرید و فلکس‌باکس)
- جاوااسکریپت (ES6+)
- Git و GitHub Pages
